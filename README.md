# Spring MVC Tutorial | Spring Boot 

## Requirements

- Java 17
- Maven

## Build Command:

```bash
mvn clean install
```

## Run Command:

```bash
./mvnw spring-boot:run 
```

## The app will start running at :

```bash
http://localhost:8080
```
